== Vidrohi Kavi

This is source code for a website dedicated to Ramashankar Yadava 'Vidrohi', a poet who spent all his life in JNU campus, New Delhi. He passed away in December, 2015. I came to know about him after his death via discussions over social media. This site is an attempt to keep multiple news, videos, poems, resources related to him at one place. 


- Manoj
